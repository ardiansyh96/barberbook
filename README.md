## 📖 About

**BarberBook** merupakan aplikasi reservasi barbershop berbasis **Flutter** yang memudahkan pelanggan melakukan booking secara online serta membantu admin mengelola operasional barbershop.

Aplikasi ini menggunakan **Firebase Authentication**, **Cloud Firestore**, dan **Firebase Storage** sebagai backend sehingga seluruh data tersimpan secara realtime.

# 🚀 Installation

Clone project

```bash
git clone https://github.com/USERNAME/barberbook.git
```

Masuk folder project

```bash
cd barberbook
```

Install package

```bash
flutter pub get
```

Jalankan project

```bash
flutter run
```
